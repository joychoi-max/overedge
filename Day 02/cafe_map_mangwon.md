## 망원동 카페 지도 (텍스트 버전)

```
[망원역 2번 출구]
       │
       │ 도보 ~3분
       ▼
  [망리단길 메인 골목]
  ├── 🏪 피피커피 (융드립 전문 · 20년 터줏대감)
  ├── 🏪 올웨이즈 어거스트 로스터스 (스페셜티 로스터리)
  ├── 🏪 딥블루레이크 (망원 대표 로스터리)
  └── 🏪 Uig
       │
       │ 도보 ~5~6분
       ▼
  [망원동 골목 안쪽]
  ├── 🏪 에프비커피로스터스 (원두 직접 로스팅)
  ├── 🏪 카페 안지
  ├── 🏪 모을
  ├── 🏪 커차그
  ├── 🏪 필담
  └── ★ 모닝브루 [우리 가게 — 골목 끝]
       │
       │ 망원시장 · 망원한강공원 방면
       ▼
```

> 🔴 위 지도는 검색 기반 추정 배치입니다. 실제 위치는 카카오맵 현장 확인 필수 ❓

## 주요 경쟁 카페 상세

### 1. 피피커피 (PP Coffee)
| 항목 | 내용 | 근거 |
|------|------|------|
| **업력** | 약 20년 (망원동 터줏대감) | 🟢 블로그 다수 언급 |
| **특징** | 융드립(Flannel Drip) 전문 · 오래된 단골 문화 | 🟢 인스타·블로그 확인 |
| **다이닝코드 평점** | ★4.7 (망원동 핸드드립 1위) | 🟢 다이닝코드 공식 |
| **분위기** | 망원역→한강 경로 · 비교적 한적 | 🟢 검색 결과 |
| **가격** | ❓ 현장 확인 필요 | 🔴 ❓ |
| **영업시간** | ❓ 현장 확인 필요 | 🔴 ❓ |

**모닝브루와의 관계:**
> 🟡 **가장 강력한 직접 경쟁자.** 20년 업력의 융드립 전문점. 단골층이 두텁고 인지도 높음.
> 핵심 차이: 모닝브루는 **당일 스콘 + 1인 좌석 + 이른 오픈(08:00)** 으로 차별화 가능. ❓

| 항목 | 내용 | 근거 |
|------|------|------|
| **특징** | 스페셜티 로스터리 · 원두 직접 선택 가능 | 🟢 다이닝코드 |
| **다이닝코드 평점** | 맛4.5 / 가격3.6 / 친절4.5 (망원동 3위) | 🟢 다이닝코드 공식 |
| **영업시간** | 월~토 12:00~22:00 · 일요일 ❓ | 🟢 검색 결과 (일 확인 필요) ❓ |
