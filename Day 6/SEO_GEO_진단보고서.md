## 2. SEO 점검 — 검색 엔진 5대 요소

### 2-1. `<title>`

| 구분 | 내용 |
|------|------|
| **현재값** | `망원동 핸드드립 카페 모닝브루 — 직접 내리는 스페셜티 커피` |
| **글자수** | 약 33자 (권장 50~60자 이내 ✓) |
| **키워드** | 지역(망원동) + 업종(핸드드립 카페) + 상호(모닝브루) ✓ |
| **평가** | ✅ 양호 — 키워드를 앞에 배치, 잘리지 않는 안전 길이 |
| **개선 여지** | 차별화 키워드 "직접 내리는" → "매일 직접 로스팅" 검토 가능 |

---

### 2-2. `meta description`

| 구분 | 내용 |
|------|------|
| **현재값** | `망원역 도보 6분, 매일 아침 직접 내리는 핸드드립 커피 전문점. 싱글 오리진 원두와 당일 구운 수제 스콘. 화~일 08:00~20:00, 온라인 픽업 예약 가능.` |
| **글자수** | 약 75자 (모바일 기준 ✓, 데스크탑 권장 150자엔 여유 있음) |
| **키워드** | 망원역, 핸드드립, 싱글 오리진, 픽업 예약 ✓ |
| **클릭 유도** | "온라인 픽업 예약 가능" — 행동 유도 문구 포함 ✓ |
| **평가** | ✅ 양호 — Day 6 적용 전 **없었음** → 추가 완료 |
| **개선 여지** | 데스크탑 검색 대비 100자 이상으로 늘려 차별점 1개 더 추가 권장 |

---

### 2-3. `<h1>`

| 구분 | 내용 |
|------|------|
| **현재값** | `모닝브루` (hero-title 클래스) + `망원동 핸드드립 카페 — 매일 아침, 직접 내리는 한 잔으로 하루를 시작하세요` (hero-sub) |
| **h1 개수** | 1개 ✓ |
| **키워드** | h1 태그 텍스트는 "모닝브루"만 → 검색 엔진이 읽는 h1에 지역·업종 없음 ⚠️ |
| **평가** | △ 개선 필요 — h1 자체에 키워드 직접 포함 권장 |
| **권장 수정안** | `<h1>망원동 핸드드립 카페 <em>모닝브루</em></h1>` |

> **왜 문제인가?** 검색 엔진은 `<h1>` 태그 텍스트를 읽습니다. CSS로 꾸민 hero-sub 문장은 참고하지만, h1 자체가 "모닝브루"만이면 핵심 키워드가 약해집니다.

---

### 2-4. 이미지 `alt` 속성

| 구분 | 내용 |
|------|------|
| **현황** | `<img>` 태그 없음 — 이모지와 CSS 배경으로만 구성 |
| **평가** | △ 해당 없음이지만 실제 사진 미등재가 SEO 기회 손실 |
| **권장 사항** | 카페 내부 사진 1~2장, 대표 메뉴 사진 추가 시 alt 필수 작성 |
| **alt 예시** | `alt="모닝브루 카페 내부 바 좌석 — 망원동 핸드드립 카페"` |
